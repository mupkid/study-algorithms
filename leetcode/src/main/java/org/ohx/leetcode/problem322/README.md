# [322. 零钱兑换](https://leetcode.cn/problems/coin-change)

## DP

假设用 i 表示数组的前 i 种硬币，用 c 表示用这 i 种硬币组成金额 c，`dp[i][c]`表示用前 i 种硬币组成金额 c 的最少硬币数，那么有该转移方程：

`dp[i][c] = min(dp[i - 1][c], dp[i][c] + 1)`

类比完全背包问题
* i 就是前 i 种物品，等同前 i 种硬币
* c 就是背包容量，等同组成金额 c
* 1 就是对应物品的价值
